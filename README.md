# calculator
<p>Calculcadora criada a partir do curso de VUE-JS da COD3R.</p>
<p>Modificações em relação ao curso original</P>
<p>1º - Representação do "0." no display. </p>
<p>2º - Representação do estado de "overflow" para  numeros com mais de 12 digits incluindo o "."</p>
